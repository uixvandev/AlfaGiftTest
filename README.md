
# Alfagift iOS Developer Technical Test

This repository contains the solution for the Alfagift iOS Developer Technical Test. The project is developed using Swift 5 and UIKit, following the provided requirements and user stories.


## Features

1. Login Feature
- Two input fields: Username and Password.
- A login button.
    - Validations:
        - Username and password cannot be empty.
        - Only valid credentials (Username: "alfagift-admin" &      Password "asdf") can log in.

2. Display List of Student Feature
-   Displays a list of 10 students.
-   Each student has name, profile pictures and address


        


## Tech Stack

**Swift 5 & UIKit**
